## Release History

<Update label="2024-10-15" description="v2.1.0" tags={["feature", "improvement"]}

## New Features

- Introduced custom domain support for branded documentation sites
- Added advanced search with filters and sorting options
- Implemented real-time collaboration features for team editing

## Improvements

- Enhanced API performance with optimized database queries
- Improved user interface with better accessibility and mobile responsiveness
- Updated documentation editor with richer formatting options

</Update>

<Update label="2024-09-20" description="v2.0.0" tags={["feature", "breaking"]}

## New Features

- Complete API redesign with RESTful endpoints and improved authentication
- Added support for multiple project workspaces
- Introduced webhook system for real-time integrations

## Breaking Changes

- API v1 endpoints are deprecated; migrate to v2 by October 15, 2024
- Authentication now requires API keys instead of basic auth
- Webhook payload format has changed; update your handlers accordingly

</Update>

<Update label="2024-08-10" description="v1.5.0" tags={["feature", "security"]}

## New Features

- Added user role management with granular permissions
- Implemented OAuth 2.0 integration for third-party logins
- Introduced document versioning and change tracking

## Security Updates

- Fixed potential XSS vulnerabilities in the editor
- Enhanced encryption for stored API keys and sensitive data
- Added rate limiting to prevent API abuse

</Update>
